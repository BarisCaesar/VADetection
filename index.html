About Project

Ventricular Arrythmias(VAs) are the main reason of Sudden Cardiac Death(SCD) but SCD can be prevented by Implementable Cardioverter Defibrillators(ICDs) if VAs can be detected in-time. Our goal in this project was to provide a deep learning model, specifically a 1D-CNN model, that can detect the arrythmias with high accuracy and efficiency. Since this model is going to be deployed on an embedded device, We have also applied some methods such as quantization to reduce the size of our model while keeping its accuracy high.
